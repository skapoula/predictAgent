# Viavi Dataset

## Where
The two files of this dataset are stored here:
- `CellReports.csv`
- `UEReports-flow.csv`

## How
Download the dataset. Files are **password protected**. Whoever is interested in using the data can comment on this page and will receive the file password via email.

## Details
- The dataset contains synthetic data generated by the simulator, consisting of two files: **Cell metrics** and **UE metrics**.
- Each file contains various metrics specific to either cells or user equipment (UE).
- Data collection occurred over a 7-day period, from **January 1, 2023** to **January 7, 2023**.
- The 4G cells use frequencies **B2** and **B17**, while the 5G cells use frequency **N77**, distributed randomly within a **10 km²** simulation area using default scenario settings.
- The data is timestamped in **Unix** format.

## Cell Metrics
| Metric | Description | Unit |
|---|---|---|
| `DRB.UEThpDl` | Downlink throughput | Gbps |
| `DRB.UEThpUl` | Uplink throughput | Gbps |
| `RRU.PrbUsedDl` | Downlink Physical Resource Blocks (PRBs) used | N/A |
| `RRU.PrbUsedUl` | Uplink Physical Resource Blocks (PRBs) used | N/A |
| `RRU.PrbAvailDl` | Number of PRBs available for downlink | N/A |
| `RRU.PrbAvailUl` | Number of PRBs available for uplink | N/A |
| `RRU.PrbTotUl` | Total usage of PRBs on uplink | % |
| `RRU.PrbTotDl` | Total usage of PRBs on downlink | % |
| `RRC.ConnMean` | Mean number of UEs in RRC connected mode | N/A |
| `RRC.ConnMax` | Maximum number of UEs in RRC connected mode | N/A |
| `QosFlow.TotPdcpPduVolumeUl` | Uplink data volume (PDCP PDU) DU → CU | Mbits |
| `QosFlow.TotPdcpPduVolumeDl` | Downlink data volume (PDCP PDU) CU → DU | Mbits |
| `PEE.AvgPower` | Average power utilization | W |
| `PEE.Energy` | Energy utilization | kWh |
| `RRU.MaxLayerDlMimo` | Avg maximum scheduled MIMO layers in DL | N/A |
| `CARR.AverageLayersDl` | Avg value of scheduled MIMO layers per PRB in DL | N/A |
| `RRC.ConnEstabAtt.mo-Data` | UE RRC connections by "data" cause | N/A |
| `RRC.ConnEstabAtt.mo-VoiceCall` | UE RRC connections by "voice call" cause | N/A |
| `RRC.ConnEstabAtt.mo-VideoCall` | UE RRC connections by "video call" cause | N/A |
| `RRC.ConnEstabSucc.mo-Data` | Successful UE RRC connections by "data" cause | N/A |
| `RRC.ConnEstabSucc.mo-VoiceCall` | Successful UE RRC connections by "voice call" cause | N/A |
| `RRC.ConnEstabSucc.mo-VideoCall` | Successful UE RRC connections by "video call" cause | N/A |
| `RRC.ConnEstabFailCause.NetworkReject` | Unsuccessful UE RRC connections rejected by network | N/A |

## UE Metrics

|---|---|---|
| `RRU.PrbUsedUl` | Mean uplink PRBs | N/A |
| `RRU.PrbUsedDl` | Mean downlink PRBs | N/A |

| `DRB.UEThpDl` | Downlink throughput | Gbps |

| `DRB.UECqiDl` | UE's downlink CQI | N/A |
